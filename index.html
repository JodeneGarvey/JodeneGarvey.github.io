Netflix Data Analysis Project

Tools & Skills: Python, Pandas, NumPy, Plotly, TextBlob, Data Cleaning, Exploratory Data Analysis (EDA), Sentiment Analysis

This project explores the Netflix Movies and TV Shows dataset to uncover meaningful patterns in content distribution, ratings, release trends, and genres. Using Python and interactive Plotly visualizations, I analyzed how Netflix’s library has evolved over the years, which countries produce the most content, and the differences between movie and TV show releases. In addition to traditional EDA, I applied sentiment analysis with TextBlob to evaluate the tone of content descriptions, categorizing them into positive, negative, and neutral sentiment. This added a natural-language perspective to the dataset, revealing how Netflix positions its content through descriptive language. Overall, the project demonstrates strong skills in data wrangling, visualization, and NLP techniques, transforming raw data into clear and engaging insights.
